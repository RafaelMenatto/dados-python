# dados-python
Script Python usando Streamlit para criar um dashboard interativo de salários na área de dados. Ele carrega um CSV online, oferece filtros laterais (ano, senioridade, contrato) e exibe KPIs de remuneração. Inclui quatro gráficos Plotly (barras, histograma, pizza e mapa) e uma tabela final, permitindo análise visual detalhada dos dados filtrados.
